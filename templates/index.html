欢迎使用论文写作智能体

一个基于大语言模型的全流程学术写作辅助工具

第一步：配置 API Key

请将您的 Google API Key 粘贴到下方输入框中。密钥将安全地存储在您的浏览器本地，不会上传到任何服务器。

开始您的研究

配置好密钥后，请从左侧菜单中选择一个功能开始。建议的流程如下：

  1. 单篇文献分析：将您的PDF文献放入 `papers` 文件夹，然后开始自动分析。
  2. 综合文献分析：选择已分析的文献，生成一份全面的文献综述报告。
  3. 头脑风暴：基于综述报告，激发创新的研究课题。
  4. 论文写作：开始撰写论文的各个章节。
